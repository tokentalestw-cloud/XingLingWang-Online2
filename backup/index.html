進階規則說明

正式回合流程

抽牌 / 召喚階段 → 二選一：戰術佈陣 或 進攻宣言 → 結算階段 → 回合結束。

戰術佈陣與進攻宣言不會同時存在。召喚階段結束後只能選一個。

目前支援的效果引擎

可支援 onSummon、onAttack、onDestroy、passiveScore。

系統會依卡片效果文字粗略轉換，例如「抽1張」、「抽2張」、「加分」、「前排+1」。
