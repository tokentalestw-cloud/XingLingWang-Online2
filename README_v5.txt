星靈王 藝術品修正版 v5

修正內容：
1. 壓縮檔重新打包，移除亂碼檔名。
2. 選擇「藝術品」後，起手一定只會從藝術品牌庫抽牌。
3. 開局起手抽4張。
4. 每次我方新回合開始抽2張。
5. 對手卡片也補齊 image/img/image_url 圖片欄位。
6. 保留原本 FastAPI / static / data 架構。

啟動方式：
1. 安裝 Python 依賴：pip install -r requirements.txt
2. 執行：uvicorn app:app --host 0.0.0.0 --port 8000
3. 開啟：http://127.0.0.1:8000


一鍵啟動方式：
- 直接雙擊「一鍵啟動_自動開網址.bat」
- 如果瀏覽器開太快導致頁面還沒載入，改用「一鍵啟動_延遲開網址.bat」
